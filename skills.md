# Shared Skills

- `react-native`: repo-local React Native router skill for CoupleGoAI, designed to layer on top of the installed React Native skill with repo-specific constraints.
- `frontend-design`: high-quality visual design skill for screens, landing surfaces, and UI polish.
- `supabase-postgres-best-practices`: Supabase-maintained Postgres guidance for schema, indexes, RLS, and performance.
- `find-skills`: discover additional skills only when explicitly requested.
- `commit-slices`: split a diff into coherent commits on request.
